# Printed Dipole Antenna Design at 1.2 GHz

**King Abdulaziz University | EE424 | 2025**

## Overview
Designed and simulated a printed dipole antenna operating at 1.2 GHz. The design utilizes a Rogers RT/duroid 5880 substrate with a dielectric constant of 2.2 and a height of 0.5 mm.

## Methods Implemented
- Analytical calculation of dipole dimensions (Width & Length)
- 3D Electromagnetic Simulation of the antenna
- Parametric study on the gap between poles for S11 optimization

## Results

| Parameter | Simulated Result |
|--------|----------|
| Operating Frequency | 1.15 GHz |
| Gain | 2.19 dBi |
| Directivity | 2.19 dBi |
| 3 dB Angular Width | 80.6 deg |
| Optimal Gap for S11 | 3 mm |

## Tools
- CST Studio Suite (3D EM Simulation)
- Theoretical Hand Calculations
